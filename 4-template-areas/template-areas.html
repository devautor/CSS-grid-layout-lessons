Nested grid layout

Named grid areas can also be used for placing items on the grid.
Define a named area and then specify the location of that area in the value of the grid-template-areas property. (See my positioning)
Line-based and named grids styles can be mixed!
In named areas method, every placement can be specified in the grid container, without touching individual items.
This enables to to make changes at different breakpoints using media queries.
Constraints:
  • a grid area can only be rectangular (yup, no making Ls)
  • grid-template-areas must have the same number of cells for each row (leave empty with a dot!)
Named areas makes it very simple to define different grid layouts on different media query breakpoints. Test the responsiveness of this page's layout vs. it size.
