The Grid Track

A grid track is the space between any two lines on the grid.
Rows & columns are defined with the grid-template-columns and grid-template-rows properties.
fr unit represents a fraction of the available space in the grid container.
For instance, this grid cell content is deciding the larger-than-required length of each explicit row by setting 1fr to half of its size.

Implicit grid
Grid implicitly creates rows & columns to place something outside of the defined grid. By default, size is auto-fit to content. Size of implicit grid can be explicitly set with the grid-auto-rows and grid-auto-columns properties. (For instance, watch sizes of the last 2 rows; these are implicit)
