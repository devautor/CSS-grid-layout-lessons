Do you observe that grid-gap is between grid-areas only, and not between grid container border and touching areas (use padding for that).

All of the boxes on the left, i.e box A, B, C and D, have a grid area of 3 columns and 3 rows (we're counting the gutter col/row). Or, 2 grid tracks by 2 grid tracks. So each box covers the same size area as box A.

The align-self property is used to align the content inside the grid-area.

You know, I am implicitly placed by auto-placement algorithm. That is, our positions in the grid are not explictly defined, and so the flow, after placing explicitly defined areas, goes through the HTML semantic, and decides where we land up. Pretty cool world you live in!
