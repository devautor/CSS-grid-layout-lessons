The Grid Container

CSS Grid Layout introduces a 2D grid system to CSS. Grids can be used to layout major page areas or small user interface elements.
A grid container is created by declaring display: grid or display: inline-grid on an element.
All direct children of that element will become grid items.
Auto-placement algorithm lays out grid-children-elements, one in each grid cell.
Grid creates a single column grid for the items unless explicitly specified. Extra rows are created as needed (implicit grid).
Gutters or alleys (i.e. margins) between grid cells can be created using the grid-column-gap and grid-row-gap properties, or the shorthand grid-gap.
